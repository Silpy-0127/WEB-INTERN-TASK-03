#  Book API

A simple REST API to manage a list of books using Node.js and Express.

##  Features
- List all books (GET /books)
- Add a book (POST /books)
- Update a book (PUT /books/:id)
- Delete a book (DELETE /books/:id)

##  Tools
- Node.js
- Express.js
- Postman
- VS Code

##  Getting Started
```bash
npm install
node index.js
